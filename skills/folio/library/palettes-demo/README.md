# Palette demo gallery

Open `index.html` in a browser to see the six palettes rendered each in their native register:

- **Telex** as an interoffice memo with priority stamp
- **F-91W** as a Casio LCD readout with hover backlight
- **Shoji** as a Bashō haiku card with vermilion seal
- **Riso Bisque** as a halftone-dotted zine flyer
- **Night Tube** as a wayfinding sign with sodium lamp glow
- **Kodachrome** as a slide mount with three saturated landscape bands

Each card shows the palette's six tokens (hover any swatch for its role), the hand-tailored specimen, the rules, and the use_for / avoid_for declarations.

The gallery is a single self-contained HTML file — no build, no dependencies. Use it as a visual reference when picking a palette during folio's Plan phase.
